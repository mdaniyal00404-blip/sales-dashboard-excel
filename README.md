# Sales Dashboard in Excel

This project is a Sales Dashboard created in Microsoft Excel to analyze business sales performance and key trends.

## Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

## Dashboard Insights
- Total Sales
- Quantity Sold
- Average Order Value
- Most Selling Category
- Order Status Analysis
- Payment Method Analysis
- Acquisition Source Performance
- Product Category Sales
- City-wise Sales and Quantity Analysis

## Project Summary
The goal of this project was to clean, organize, and visualize sales data in a clear and interactive dashboard. This dashboard helps understand sales performance, customer order behavior, product category performance, and acquisition source results.
